## 项目运行

> Vue 不支持 IE8 及其以下版本，因为 Vue.js 使用了 IE8 不能模拟的 ES5 特性。
> Vue.js 支持所有兼容 ECMAScript 5 的浏览器。

```
git clone https://github.com/Virgo9/bigBoss.git
cd bigBoss-web/bigBoss-web-api/src/main/webapp/public/
npm install

```
